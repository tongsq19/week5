# Spring boot 基础

项目由 gradle 构建

主要代码在 src/main/java/com.example.homework 目录下

运行方法：

在项目根目录下使用 `./gradlew bootRun` 命令启动服务器，

出现 `Tomcat started on port(s): 8080 (http)` 字样，说明项目启动成功
